Activities for the **Fundamentals of Programming (FDP)** course at the Faculty of Engineering, UNAM.

## 📘 What You'll Find Here:

- 💻 **In-class activities** and introductory exercises in C.
- 🧪 **Lab practice reports** (Practice 1 – 12).
- 📝 **Problem series** for the first and second partials.
- 🎬 **Video Project** presentation.
- 📁 **Final Project** for the course.

## 🛠️ Technologies & Applied Concepts

- **C language** — First programming language in the study plan.
- **Data types, variables, and operators.**
- **Control structures:** `if`, `while`, `do-while`, `for`.
- **Functions** and modularization.
- **Arrays and matrices.**
- **Text file handling** (`.txt`).
- **Flowcharts** for pseudocode design.
